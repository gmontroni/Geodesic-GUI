|COMMANDS GEODESIC GUI                      |
|-------------------------------------------|
|left-click: source point geodesic.         |
|right-click: destination point geodesic.   |
|w: determine the base mesh.                |
|r: removes the last created geodesic path. |
|s: save arquive json and off.              |
|f: hides the face.                         |
|F: shows the face.                         |
|e: hides the edge.                         |
|E: shows the edge.                         |
